■ エグゼクティブ・サマリー

資産運用のリターンの約93.6%は
アセットアロケーション（資産配分）で決定される

  • 銘柄選定・タイミングは補助要素
  • 暴落時の最適戦略＝リバランス
  • 長期投資の鍵＝リスク許容度＋家計力

2️⃣ 暴落局面の戦略：リバランス

例：50%株 / 50%現金

① 初期：株1000万 / 現金1000万

② 下落後：株800万 / 現金1000万

③ リバランス：現金100万 → 株購入

⇒ 株900万 / 現金900万（比率回復）

✔ 安く買い・高く売るを自動実行

📉 暴落時

安全資産から取り崩す

📈 上昇時

リスク資産から取り崩す

3️⃣ リスク管理と心理

理論上

株100%が最大期待値

現実

暴落で耐えられない → 売却

結論：
守れない最適解 ＜ 守れる配分

事前準備：
40〜50%下落を想定して損失額を把握

相関分散：
株 × 債券（相関 -0.28）
→ リスク低減
