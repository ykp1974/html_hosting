3. 進化した5大機能

MCP (連携)

Gmail、カレンダー、Notionなど外部ツールと「脳」を接続。

スキル (学習)

独自の知識やマニュアルを学習させ、特定のトーンやルールを守らせる。

プラグイン (拡張)

ブラウザ操作や動画生成など、特化機能をプラグイン形式で追加可能。

サブエージェント

内部で専門特化型AIを自律運用し、複雑なタスクを分業処理。
