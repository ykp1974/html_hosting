4️⃣ 高度な機能：MCPと「スキル」の活用

🔌 MCP（Model Context Protocol）

外部サービスと接続するための共通規格

  • Notion、Googleカレンダーなどのデータを直接読み取り可能
  • APIがあれば、接続設定をAI自身に作らせることもできる
  • 様々なツールを統合的に管理

💡 「スキル」による業務のパッケージ化

一連の処理をマニュアル化して保存する仕組み

  • 作成方法： 作業を一度実行 →「今の手順をスキル化して」と指示
  • 再利用： スキル名を呼ぶだけで同じ処理を再現
  • 中身の自動生成： 自然言語やコードはAIが自動で設定・更新
