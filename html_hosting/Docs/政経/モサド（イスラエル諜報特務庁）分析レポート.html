イスラエル諜報特務庁「モサド」分析レポート

[モサド] 世界で最も恐れられるスパイ組織、想像以上にヤバすぎた...

エグゼクティブ・サマリー

モサドは、イスラエル首相の直轄下で国家の存亡を左右する極秘工作を担う組織。軍や議会の監視から独立した超法規的権限を持ち、武力だけでなく心理戦と虚偽の構築を「武器」として、世界地図を自国の利益に沿って書き換える能力を有しています。

2. 組織の特権的地位

首相直属の独立性

防衛大臣の管轄を外れ、首相と長官のホットラインで直結。閣議や議会の監視を必要としない極めて閉鎖的な権限体系。

サヤニム（Sayanim）

世界中に存在するユダヤ系協力者ネットワーク。公的記録に残らない形で隠れ家や車両を提供し、工作を無償支援する「見えない手」。

3. 徹底した心理工作手法

「嘘」という最強の武器

教育の根幹は「銃」ではなく「嘘」にある。数日間、別人を演じ切る高度な演技力と、感情を完全に遮断する冷徹さが工作員の資質。

痕跡の消去

現地の部外者や無自覚な協力者を介在させることで、自らの痕跡を消し、実行犯が「誰なのか」すら分からなくさせる巧妙な工作。

4. 主要作戦の衝撃

2018年：イラン核密文書強奪作戦

テヘランの倉庫から500kg超の機密資料を強奪し、それを会見で「演出」して公開。この1週間後に米国がイラン核合意からの離脱を決定する等、情報公開を外交の武器として最大化した。

テクノロジーによる心理支配： AI補正兵器や通信機器の同時爆発など、神出鬼没の攻撃を通じて「どこにいても逃げられない」という圧倒的絶望感を植え付ける。

結論：情報の「収集」ではなく、情報を「動かすための道具」として活用することで世界を動かす組織である。
